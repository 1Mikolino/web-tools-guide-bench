# T01 — web_search：无 URL 搜索

## 任务描述
用户没有指定 URL，需要搜索「上海今天天气」。

## 预期决策路径
```
无 URL？
  └─ YES → web_search("上海今天天气")
        └─ Tavily API 可用 → 成功返回结果
```

## 执行步骤
1. Agent 判断：无 URL → 调用 `web_search`
2. `web_search` 通过 Tavily API 执行搜索
3. 返回搜索结果（标题 + URL + 摘要）

## 成功标准
- [ ] 返回至少 1 条搜索结果
- [ ] 结果中包含「天气」相关关键词
- [ ] 耗时 ≤ 5s
