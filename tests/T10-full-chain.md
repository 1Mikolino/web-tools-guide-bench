# T10 — 全链路：搜索 → 多结果抓取 → 对比

## 任务描述
完整链路：搜索「Redis 缓存最佳实践」→ 对前 2 个结果 URL 分别 `web_fetch` → 对比两篇文章的摘要。

## 预期决策路径
```
web_search("Redis 缓存最佳实践")
  → results[0] → web_fetch(url0)
  → results[1] → web_fetch(url1)
  → 对比两篇内容摘要
```

## 执行步骤
1. `web_search("Redis 缓存最佳实践", count=5)`
2. 取 `results[0].url` 和 `results[1].url`
3. 并行（或串行）`web_fetch(url0)` 和 `web_fetch(url1)`
4. 对比两篇文章的：
   - 标题
   - 内容摘要（前 200 字）
   - 发布时间（若有）
5. 输出对比表格

## 成功标准
- [ ] web_search 返回 ≥ 2 条结果
- [ ] 两个 URL 均抓取成功
- [ ] 对比表格包含 ≥ 3 个对比维度
- [ ] 总耗时 ≤ 20s
